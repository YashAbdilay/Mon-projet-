# Web Innovation — Portfolio (HTML/CSS/JS)

Un site vitrine professionnel, responsive et esthétique pour un groupe de 5 étudiant·e·s. Le titre principal (H1) est « Web innovation » comme demandé.

## Démarrage

- Ouvrir `index.html` dans votre navigateur.
- Aucun build requis (pur HTML/CSS/JS).

## Personnalisation rapide

- Équipe: modifier les cartes dans `index.html` (section `#equipe`).
- Projets: dupliquer/éditer les `<article class="project-card">` dans `#projets` et ajuster `data-category`, `data-title`, `data-description`, `data-image`.
- Compétences: éditer la liste `#competences`.
- Couleurs/typographies: ajuster les variables CSS dans `styles.css` (section `:root`) et les polices Google dans le `<head>`.

## Fonctionnalités

- Thème clair/sombre avec persistance (bouton dans le header).
- Filtres par catégories de projets.
- Modale de détails de projet.
- Animations d’apparition au scroll.
- Formulaire de contact avec validation côté client et toast.

## Accessibilité

- Sémantique, rôles ARIA sur la modale, focus non piégé volontairement pour simplicité (peut être amélioré).
- Couleurs à contraste renforcé et tailles responsives.
